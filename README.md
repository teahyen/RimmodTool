# 🧬 RimWorld Biotech Xenotype Builder 사용 설명서

본 도구는 림월드(RimWorld)의 **Biotech DLC** 유전자 시스템을 기반으로 한 새로운 종족(제노타입) 모드를 브라우저에서 편리하게 제작할 수 있도록 돕는 도구입니다.

---

## 🛠️ 필수 요구 사항

- **RimWorld 본편**
- **Biotech DLC** (필수)

---

## 🚀 사용 순서

### 1. 기본 정보 입력

- **모드 이름 & 제작자**: 게임 내 모드 리스트에 표시될 이름과 당신의 아이디를 입력합니다.
- **제노타입 ID (DefName)**: 시스템에서 인식할 고유 코드입니다. 영문으로 작성하세요. (예: `MySuperHuman`)
- **표시 이름**: 게임 내에서 실제로 보게 될 한글/영문 이름입니다.

### 2. 외형 및 스탯 설정

- **피부/머리카락 색상**: 컬러 피커를 통해 종족의 고정된 색상을 지정할 수 있습니다.
- **스탯 유전자**: 이동 속도, 체력 배율, 허기율 등을 조정합니다. 조정된 값에 따라 자동으로 유전자가 생성됩니다.

### 3. 특수 부위 및 끔찍한 연구 (핵심)

- **특수 부위**: 추가 심장, 발톱, 독 분비샘, 날개 등 신체 구조를 변경하는 강력한 유전자를 선택합니다.
- **끔찍한 연구**: 혈액 흡식(뱀파이어), 불사 회복, 도살 욕구 등 일반적인 인류를 벗어난 극단적인 능력을 부여합니다.
- **강제 특성**: 강인함, 사이코패스 등 종족 전체가 공유할 고정 특성을 선택합니다.

### 4. ⚖️ 대사 균형 (Metabolism) 체크

- 우측 상단의 **플로팅 바**를 확인하세요.
- 강력한 유전자를 선택할수록 대사가 낮아져(음수) 캐릭터가 더 많은 음식을 섭취해야 합니다.
- 반대로 부정적인 유전자를 선택하면 대사가 높아져(양수) 효율적인 생존이 가능합니다.

### 5. 아카이브 및 텍스처

- **아이콘**: 제노타입을 상징할 128x128 PNG 이미지를 업로드하세요.
- **모드 생성**: 최하단의 **[🧬 제노타입 모드 생성]** 버튼을 누르면 즉시 `.zip` 파일이 다운로드됩니다.

---

## 📂 게임 적용 방법

1. 다운로드된 `.zip` 파일의 압축을 풉니다.
2. 폴더를 림월드 모드 경로에 복사합니다.
   - **스팀 버전**: `C:\Program Files (x86)\Steam\steamapps\common\RimWorld\Mods`
3. 게임을 실행하고 **[모드(Mods)]** 메뉴에서 제작한 모드를 활성화합니다.
4. 게임 내 '제노타입 편집' 또는 '시나리오 설정'에서 당신이 만든 종족을 찾아보세요!

---

## ⚠️ 주의 사항

- 본 도구는 **순정(Vanilla) Biotech XML** 구조를 사용하므로 다른 모드와의 충돌이 적습니다.
- 아키타이트(Archite) 등급 유전자는 게임 내 밸런스를 파괴할 수 있으니 신중히 사용하세요.
